# Rollups Notes

## Layer 2 Rollups

Rollups improve Ethereum scalability.

## Benefits

- Lower gas fees
- Faster transactions
- Better scalability

## Examples

- Base
- Optimism
- Arbitrum
## Rollup Types

There are two common rollup categories:

- Optimistic rollups
- Zero-knowledge rollups

Rollups help Ethereum scale by processing transactions off the main chain.
