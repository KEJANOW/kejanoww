# Base Wallet Guide

## Getting Started

1. Install Coinbase Wallet.
2. Add Base Network.
3. Bridge ETH to Base.
4. Explore Base applications.

## Tips

- Keep recovery phrase offline.
- Verify official links.
- Use trusted bridges.
## Wallet Connection Tips

Before connecting a wallet:

- Verify the website.
- Check the network.
- Read requested permissions.
- Disconnect unused applications regularly.
