# Smart Contracts

## Overview

Smart contracts are programs that run on blockchain networks.

## Key Features

- Automatic execution
- Transparent rules
- Immutable after deployment

## Learning Goals

- Understand Solidity
- Learn contract deployment
- Explore Base ecosystem
## Smart Contract Lifecycle

1. Write code
2. Compile
3. Deploy
4. Verify
5. Interact
6. Upgrade when needed
